# 🦈 Shark Tank India Analysis Dashboard - Power BI

## 📌 Project Overview
This project is an interactive **Shark Tank India Dashboard** built using Power BI.  
It analyzes startup pitches, investment patterns, and shark behavior to uncover meaningful business insights.

---

## 🎯 Objectives
- Analyze startup funding trends
- Understand investor (sharks) behavior
- Identify top industries and successful pitches
- Evaluate deal success rates

---

## 📂 Dataset Features
The dataset includes:
- Startup Name
- Industry / Domain
- Investment Amount
- Equity Offered
- Deal Status (Deal / No Deal)
- Sharks Involved
- Season & Episode details

---

## 📊 Dashboard Insights
Key insights covered:
- 💰 Total Investment Analysis
- 🦈 Shark-wise Investment Trends
- 📊 Deal vs No Deal Ratio
- 🏆 Top Funded Startups
- 📈 Industry-wise Funding Distribution

---

## 🛠️ Tools & Technologies Used
- Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Data Analysis Expressions)

---

## 🚀 Features
- Interactive filters and slicers
- Dynamic and visually appealing charts
- Drill-down capabilities
- Clean and professional dashboard layout
